# 📈 AI Equity Research Terminal

An AI-powered equity research platform built in Python that automates financial research by collecting market data, analyzing financial statements, summarizing earnings reports, performing company valuation, stress-testing investment theses, and generating professional investment memos.

---

## 🚀 Features

- Download financial statements
- Retrieve historical stock prices
- Company profile extraction
- SEC filing analysis (Coming Soon)
- Earnings call summarization (Coming Soon)
- DCF Valuation (Coming Soon)
- Comparable Company Analysis (Coming Soon)
- Bull / Base / Bear Scenario Analysis (Coming Soon)
- AI Investment Memo Generator (Coming Soon)
